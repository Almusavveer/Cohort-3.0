🧪 Attributes vs Properties

Explore how elements differ in their HTML source configuration (Attribute) versus their live in-memory JavaScript representation (Property).

input.value (Property)
-

Reads from the live DOM node object state. Updates in real-time as you type.

input.getAttribute("value")
-

Reads from the initial HTML markup attribute. Does NOT change automatically when typing.
